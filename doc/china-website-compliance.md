# 中国大陆网站上线合规清单

> 更新日期：2025年12月5日  
> 适用项目：医学宝典（医学考试备考平台）

---

## 一、必须办理（强制性）

### 1. ICP 备案 ⭐⭐⭐
| 项目 | 说明 |
|------|------|
| **全称** | 互联网信息服务备案 |
| **办理机构** | 通过服务器/云服务商代办（阿里云、腾讯云等） |
| **所需材料** | 域名证书、营业执照、法人身份证、网站负责人身份证、核验单 |
| **办理周期** | 约 15-20 个工作日 |
| **展示要求** | 网站底部展示备案号，链接至 https://beian.miit.gov.cn |
| **示例** | `京ICP备XXXXXXXX号` 或 `京ICP证XXXXXX号`（经营性） |
| **费用** | 免费（通过云服务商办理） |

**注意**：
- 未备案网站使用国内服务器将被屏蔽
- 个人备案 vs 企业备案：医学教育类建议使用企业主体
- 备案主体需与域名持有者一致

---

### 2. 公安备案 ⭐⭐⭐
| 项目 | 说明 |
|------|------|
| **全称** | 全国互联网安全管理服务平台备案 |
| **办理网址** | https://www.beian.gov.cn |
| **办理时间** | ICP 备案通过后 30 日内完成 |
| **所需材料** | ICP 备案号、营业执照、法人身份证、网站负责人身份证 |
| **办理周期** | 约 3-7 个工作日 |
| **展示要求** | 网站底部展示公安备案号和图标，链接至备案查询页面 |
| **示例** | `京公网安备 11010502XXXXXX号`（带公安徽章图标） |
| **费用** | 免费 |

## 二、根据业务类型可能需要的资质

### 3. 增值电信业务经营许可证（ICP 许可证）⚠️
| 项目 | 说明 |
|------|------|
| **适用场景** | 网站有付费内容、会员收费等经营性行为 |
| **办理机构** | 省级通信管理局 |
| **要求** | 注册资本 100 万以上、3名以上专业人员 |
| **办理周期** | 约 40-60 个工作日 |
| **费用** | 自办免费，代办约 3000-10000 元 |
| **示例** | `京ICP证XXXXXX号` |

**医学宝典分析**：
- ✅ 如果有付费题库、会员订阅等功能，**需要办理**
- ❌ 如果完全免费，仅需 ICP 备案

---

### 4. 网络文化经营许可证（文网文）⚠️
| 项目 | 说明 |
|------|------|
| **适用场景** | 提供网络音乐、动漫、游戏、演出等文化产品 |
| **办理机构** | 省级文化和旅游厅 |
| **办理周期** | 约 20-40 个工作日 |

**医学宝典分析**：
- ❌ 教育类内容通常不需要，除非包含大量视频/动漫课程

---

### 5. 信息网络传播视听节目许可证 ⚠️
| 项目 | 说明 |
|------|------|
| **适用场景** | 提供视频、音频点播服务 |
| **办理机构** | 国家广播电视总局 |
| **难度** | 极高，通常只有大平台能获得 |

**医学宝典分析**：
- ❌ 如果只是嵌入第三方视频（如B站、腾讯视频），不需要
- ⚠️ 如果自建视频服务，建议咨询专业机构

---

### 6. 互联网出版许可证 ⚠️
| 项目 | 说明 |
|------|------|
| **适用场景** | 提供电子书、数字出版物下载 |
| **办理机构** | 国家新闻出版署 |

**医学宝典分析**：
- ⚠️ 如果提供讲义/教材 PDF 下载，可能需要
- 建议：讲义作为"学习资料"而非"出版物"定位，或仅在线阅读不提供下载

---

### 7. 教育培训相关资质 ⚠️
| 项目 | 说明 |
|------|------|
| **在线教育备案** | 教育 APP 需在教育部备案 |
| **办理网址** | https://app.eduyun.cn |
| **适用场景** | 提供在线教育/培训服务的 APP 或网站 |

**医学宝典分析**：
- ⚠️ 可能需要进行教育 APP 备案
- 建议咨询当地教育主管部门

## 四、办理流程时间线

```
第1周    ──────────────────────────────────────
         │ 准备材料（营业执照、域名证书、身份证等）
         │ 域名实名认证（如未完成）
         
第2-4周  ──────────────────────────────────────
         │ 提交 ICP 备案（通过云服务商）
         │ 等待管局审核
         │ 可能需要接听核验电话
         
第5周    ──────────────────────────────────────
         │ ICP 备案通过
         │ 立即提交公安备案
         
第6周    ──────────────────────────────────────
         │ 公安备案通过
         │ 网站正式上线
         
后续     ──────────────────────────────────────
         │ 如需经营性资质，继续申请 ICP 许可证等
```

## 五、所需材料清单

### 企业主体备案材料

| 序号 | 材料名称 | 要求 |
|------|----------|------|
| 1 | 营业执照副本 | 彩色扫描件，清晰可辨 |
| 2 | 法定代表人身份证 | 正反面扫描件 |
| 3 | 网站负责人身份证 | 正反面扫描件（可与法人同一人） |
| 4 | 网站负责人手持身份证照片 | 部分省份要求 |
| 5 | 域名证书 | 从域名注册商下载 |
| 6 | 网站备案信息真实性核验单 | 云服务商提供模板，签字盖章 |
| 7 | 授权书 | 如网站负责人非法人，需提供 |
| 8 | 服务器/主机信息 | 云服务商会自动填充 |

| 序号 | 材料名称 | 要求 |
|------|----------|------|
| 1 | ICP 备案号 | ICP 备案通过后获得 |
| 2 | 网站安全责任人信息 | 姓名、身份证、联系方式 |
| 3 | 网站开办主体信息 | 单位或个人基本信息 |
| 4 | 网站基本情况 | 服务类型、涉及内容等 |

## 六、常见问题

### Q1: 使用香港/海外服务器可以不备案吗？
- 可以，但国内用户访问速度慢，且无法使用国内 CDN
- 部分支付接口（如微信支付、支付宝）要求网站已备案

### Q2: 个人可以备案教育类网站吗？
- 可以备案，但部分省份限制个人备案的网站类型
- 建议使用企业主体备案，更合规

### Q3: 备案期间网站能访问吗？
- 首次备案：网站需关闭，或使用临时域名
- 新增备案：已备案域名的网站可正常访问

### Q4: 更换服务器需要重新备案吗？
- 同一云服务商内更换：不需要
- 更换云服务商：需要办理备案接入

### Q5: 备案信息变更怎么办？
- 公司名称、负责人等变更，需要在 30 日内办理变更备案

## 十、重要提醒

1. **先备案，后上线** - 未备案网站使用国内服务器会被屏蔽
2. **保持信息一致** - 备案信息与网站实际信息需一致
3. **定期检查** - 每年核查备案信息是否需要更新
4. **咨询专业人士** - 涉及特殊资质建议咨询律师或代办机构
